# Thomas Chadefaux - Minimalist Website

This is a new minimalist website design inspired by Olivia Montagnese's portfolio style, adapted for academic content.

## Design Features

Following the Olivia Montagnese design aesthetic:

- **Monochromatic color scheme**: Black text (#000000) on white background (#ffffff) with gray accents (#757575)
- **Clean typography**: System fonts with precise spacing (line-height: 1.33em, letter-spacing: -0.1px)
- **Minimalist layout**: Grid-based responsive design with generous whitespace
- **Fixed navigation**: Simple top navigation bar with minimal links
- **Hover effects**: Subtle opacity transitions on interactive elements
- **Responsive design**: Adapts seamlessly from desktop to mobile

## Content Sections

1. **About**: Brief bio and research focus
2. **Publications**: Grid of recent publications with links
3. **Research**: PaCE project and Forecast Lab details
4. **Contact**: Email and CV download

## How to Use

1. Open `index.html` in any web browser
2. To deploy: Upload the file to your web hosting service
3. To customize: Edit the HTML file directly - all styles are embedded

## Customization

All content and styling is contained in a single HTML file for easy deployment. To update:

- **Content**: Modify the text within the `<main>` section
- **Styling**: Adjust the CSS in the `<style>` section
- **Colors**: Change the hex values in the CSS
- **Navigation**: Update links in the `<nav>` section

## Next Steps

To add more publications or projects, simply copy one of the `.grid-item` divs and update the content with your new information.
